## Fórmula extraída

```
Frame 1-6 → Setup de situación + avatar activo (HACE cosas, no posa)
Frame 7   → PUNCH LINE WTF — el elemento más inesperado
```

### Reglas obligatorias

- **7 frames exactos** (ni 6 ni 8)
- **El objeto principal = devoción** (el Porsche / el frasco de perfume). Nunca se daña. Siempre perfecto.
- **Ella HACE cosas** con el invasor — pesca, come, sostiene. No posa.
- **Animales = elemento más viral** (oso > camello > gato > gallinas)
- **Localización épica real** — Sahara, Suiza, NYC, Barcelona. No estudios.
- **Interior destruido** = frame obligatorio (con arena, huevos, agua)
- **Caption = acción deadpan o declaración de amor absurda**. NUNCA descriptiva de la escena.

### Qué hace que "tropical vibes." sea la peor:
La caption describe la escena. El resto no describen — declaran algo absurdo o ejecutan una acción.

## Traducción al canal de perfumes

```
El frasco de perfume = el Porsche
Las notas del perfume = el elemento invasor que se materializa visualmente
```

- Perfume con nota de oud + madera quemada → frame 7: nómada del desierto aparece en el apartamento
- Perfume con nota de mar + algas → frame 7: pulpo gigante en la bañera
