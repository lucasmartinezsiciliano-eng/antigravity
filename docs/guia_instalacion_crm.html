Guía de instalación — CRM Firmax

Paso a paso para poner el CRM en los ordenadores del equipo

⚡ ANTIGRAVITY

¿Qué es esto? El CRM Firmax es un programa que vive en un solo ordenador de la oficina (el tuyo, por ejemplo) y al que todos los demás acceden desde su navegador, igual que cuando abres una web normal. No hay que instalar nada en los ordenadores de Mariano, Silvia ni Carlos — solo abrir Chrome y escribir una dirección.

1

Arrancar el servidor en tu ordenador

Solo se hace una vez

Abre la carpeta mission-control que está dentro de tu proyecto ANTIGRAVITY en el escritorio.

Haz clic en la barra de direcciones de la carpeta, escribe cmd y pulsa Enter. Se abrirá una ventana negra.

En esa ventana negra, escribe exactamente esto y pulsa Enter:

node server.js

Si todo va bien, verás este mensaje en la pantalla:

🚀 Mission Control en http://localhost:3333
📊 CRM Firmax en http://localhost:3333/crm.html
⚠️

Importante: No cierres esa ventana negra mientras trabajéis. Si la cierras, el CRM deja de funcionar para todos.

2

Encontrar la dirección de tu ordenador (la IP)

La IP es la "dirección" de tu ordenador dentro del WiFi de la oficina. Es como el número de una casa. La necesitas para que los demás sepan a dónde conectarse.

Pulsa las teclas Windows + R al mismo tiempo. Se abrirá una ventanita pequeña.

Escribe cmd y pulsa Enter.

En la ventana negra que se abre, escribe esto y pulsa Enter:

ipconfig

Busca la línea que dice "Dirección IPv4". El número que aparece ahí es tu IP.

📌 Ejemplo: Si tu IP es 192.168.1.45, la dirección del CRM para los demás será:
http://192.168.1.45:3333/crm.html

Anota aquí tu IP para tenerla a mano:

http://_____ . _____ . _____ . _____ :3333/crm.html
3

Abrir el CRM en los ordenadores de los trabajadores

Esto es lo único que tienen que hacer Mariano, Silvia y Carlos. No hay que instalar nada.

Abrir Google Chrome (o cualquier navegador).

En la barra de arriba (donde se escribe la web), escribir la dirección del CRM:

http://TU_IP:3333/crm.html

Pulsar Enter. Se abrirá el CRM.

En el menú lateral izquierdo, cada uno elige su nombre en el campo "Conectado como". Solo hay que hacerlo una vez — el navegador lo recuerda.

💡 Consejo: Diles que guarden esa dirección como marcador en Chrome para no tener que escribirla cada vez. Botón de estrella ⭐ en Chrome → "Añadir a favoritos".

4

Que el servidor arranque solo al encender el PC

Recomendado

Sin este paso, tendrías que acordarte de abrir la ventana negra y escribir node server.js cada vez que enciendes el ordenador. Con PM2, el servidor arranca solo.

Solo la primera vez — instala PM2. Abre una ventana negra (cmd) y escribe:

npm install -g pm2

Ve a la carpeta mission-control. En la barra de la carpeta escribe cmd y pulsa Enter. Luego ejecuta:

pm2 start server.js --name "crm-firmax"
pm2 save
pm2 startup

El último comando puede pedirte que copies y pegues una línea más. Hazlo y pulsa Enter.

✅ A partir de ahora el CRM se arranca solo cuando enciendes el ordenador. No tienes que hacer nada más.
Comando Para qué sirve
pm2 status Ver si el servidor está corriendo
pm2 stop crm-firmax Parar el servidor
pm2 restart crm-firmax Reiniciar el servidor
pm2 logs crm-firmax Ver si hay algún error
5

Si algo no funciona — soluciones rápidas

Problema Solución
La página no carga en el PC de un trabajador Comprueba que la ventana negra con node server.js siga abierta en tu PC. Si está cerrada, vuelve a abrirla.
Windows pide permiso de firewall Pulsa "Permitir acceso" cuando aparezca la ventana. Solo pasa la primera vez.
La IP cambió y ya no funciona Vuelve a hacer el paso 2 (ipconfig) para ver la nueva IP y dísela a los trabajadores.
El chat no me muestra mi nombre Asegúrate de haber seleccionado tu nombre en el menú lateral izquierdo del CRM.
El servidor no arranca (error en ventana negra) Asegúrate de estar dentro de la carpeta mission-control antes de escribir el comando.
